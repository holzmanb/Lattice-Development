About

Lattice is an abstract strategy game created in the back row of a geophysics class at the University of British Columbia.

Game rules were created by Ben Holzman, and tested with Garrett Miller.

AI testing and web development were done by Al Grant and Ben Holzman
Source code is on github: github.com/holzmanb/Lattice-Development
